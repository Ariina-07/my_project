Ответы на вопросы ПР5:

1. Основное свойство - display: grid. Grid работает в двух измерениях (строки и колонки), Flexbox - в одном.

2. grid-template-columns задает явные колонки, grid-auto-columns - размер автоматических колонок.

3. repeat() повторяет шаблон, minmax() задает мин/макс размер, auto-fit автоматически подстраивает колонки.

4. Именованные области (grid-template-areas) упрощают создание макетов через визуальные шаблоны.

5. Линии нумеруются с 1. grid-column: 1/3 - элемент от 1 до 3 линии.

6. gap создает равные промежутки между всеми элементами сетки.

7. justify-items выравнивает по горизонтали, align-items - по вертикали.

8. auto-fill создает пустые колонки, auto-fit растягивает существующие.

9. grid-template-areas создает макет через имена. Правила: одинаковое количество ячеек в строках.

10. Grid позволяет создавать сложные двумерные макеты без лишней HTML разметки.
