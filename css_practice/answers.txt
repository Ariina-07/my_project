Ответы на вопросы ПР3:

1. Селектор по классу имеет специфичность (0,1,0), по ID - (1,0,0). Классы можно повторно использовать, ID - только один раз на странице.

2. Каскад применяет стили в порядке: важность (!important), специфичность, порядок в коде.

3. Специфичность:
   - ul li.important = (0,1,2)
   - #site-title = (1,0,0)
   - .content > .featured-post = (0,2,0)

4. Наследуются: color, font-family, line-height. Не наследуются: margin, padding, border.

5. Селектор > выбирает только прямых потомков, через пробел - всех потомков.

6. :hover для эффектов при наведении, :nth-child() для выбора элементов по порядку.

7. !important повышает приоритет правила, но усложняет поддержку кода.

8. ::before и ::after добавляют контент до/после элемента, могут содержать текст, изображения.

9. При одинаковой специфичности применяется последнее правило в коде.

10. Инструменты разработчика показывают примененные стили, специфичность, позволяют экспериментировать.
