Ответы на вопросы для самоконтроля

1. Что такое Node.js и в чем его преимущества для создания серверов?

Node.js - это среда выполнения JavaScript на стороне сервера. Преимущества:
- Использование JavaScript и на клиенте, и на сервере
- Высокая производительность благодаря событийно-ориентированной архитектуре
- Большое количество готовых модулей (npm)
- Хорошая масштарируемость

2. Как создать простой HTTP-сервер в Node.js?

```javascript
const http = require('http');

const server = http.createServer((req, res) => {
    res.writeHead(200, {'Content-Type': 'text/plain'});
    res.end('Привет, мир!');
});

server.listen(3000);
