Анализ legacy-кода

Выявленные проблемы:
1. Смешение PHP логики и HTML представления
2. Уязвимость к SQL-инъекциям
3. Отсутствие обработки ошибок и валидации

Установка и запуск проекта

Требования:
- PHP 8.0+
- Composer

Установка:
composer create-project laravel/laravel my-project
cd my-project

php artisan serve
