=== Примеры запросов для тестирования Express.js API ===

1. Основные маршруты:

GET корневой путь:
curl -X GET http://localhost:3000/

2. Пользователи:

GET всех пользователей:
curl -X GET http://localhost:3000/api/users

GET пользователя по ID:
curl -X GET http://localhost:3000/api/users/1

POST создать пользователя:
curl -X POST http://localhost:3000/api/users \
  -H "Content-Type: application/json" \
  -d '{"name":"Новый пользователь","email":"new@example.com","age":22}'

PUT обновить пользователя:
curl -X PUT http://localhost:3000/api/users/1 \
  -H "Content-Type: application/json" \
  -d '{"name":"Обновленное имя","age":26}'

DELETE удалить пользователя:
curl -X DELETE http://localhost:3000/api/users/1

3. Продукты:

GET все продукты:
curl -X GET http://localhost:3000/api/products

GET продукты с фильтрацией:
curl -X GET "http://localhost:3000/api/products?category=Ноутбуки"
curl -X GET "http://localhost:3000/api/products?inStock=true"

GET продукт по ID:
curl -X GET http://localhost:3000/api/products/1

POST создать продукт:
curl -X POST http://localhost:3000/api/products \
  -H "Content-Type: application/json" \
  -d '{"name":"Новый продукт","price":50000,"category":"Электроника"}'

PUT обновить продукт:
curl -X PUT http://localhost:3000/api/products/1 \
  -H "Content-Type: application/json" \
  -d '{"price":160000,"inStock":false}'

DELETE удалить продукт:
curl -X DELETE http://localhost:3000/api/products/1

4. Тестирование ошибок:

Несуществующий маршрут:
curl -X GET http://localhost:3000/api/nonexistent

Некорректный JSON:
curl -X POST http://localhost:3000/api/users \
  -H "Content-Type: application/json" \
  -d 'invalid-json'
